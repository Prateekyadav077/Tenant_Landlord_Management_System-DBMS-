# Tenant Management System

A DBMS project developed using MySQL and Streamlit for managing rental properties, tenants, and rent payments.

## Features

* Normalized relational database design (3NF)
* Tenant and Landlord management
* Property allocation system
* Rent payment tracking
* SQL joins, subqueries, and views
* MySQL triggers and stored procedures
* Transaction management using COMMIT and ROLLBACK
* Landlord and Tenant dashboards

## Technologies Used

* MySQL
* SQL
* Streamlit
* Python
* Pandas

## Database Components

* Users
* Login
* Landlord
* Tenant
* Property
* Allocated_To
* Payment

## SQL Concepts Implemented

* Joins
* Subqueries
* Views
* Aggregate Queries
* Triggers
* Stored Procedures
* Transactions

## How to Run

1. Import tenant_system.sql into MySQL.
2. Install dependencies:

pip install -r requirements.txt

3. Run:

streamlit run app.py
